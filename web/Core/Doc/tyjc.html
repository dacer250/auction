一、 绑定邀请码

1. 登录代理商后台后，记录自己的邀请码

2. 让下线在招宝长沙棋牌APP中点击自己头像

3. 选择推荐人→在推荐人ID内填入你的推荐ID

4. 验证下线，绑定是否成功

1) 点右上角菜单→提现账号

2) 验证学员ID和注册时间，是不是在列表中

二、 查收入

1. 点右上角菜单→收入详情

2. 选择“开始时间”和“结束时间”，点查询，即可显示收入数据

3. 有充值的日期，会显示在列表中，点“明细”查询详情

4. 收入明细中显示下线的充值金额、层级关系、分成比例和收入金额，如内容为空说明当日用户未充值

三、 提现

1. 点右上角菜单→提现账号

2. 填写：支付宝姓名、支付宝账户，填写验证码，点保存

3. 填写短信验证码，保存提现信息

4. 静待收入到账，工作日当日到账，节假日次日到账

四、 修改密码

1. 点右上角菜单→我的信息

2. 填写新密码→填写短信验证码→保存→修改成功→重新登录验证
